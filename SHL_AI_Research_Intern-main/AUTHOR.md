## Project Overview

An intelligent recommendation system that helps hiring managers and recruiters find the right SHL assessments for their roles using natural language queries or job descriptions.

## Key Features

- Semantic search using sentence transformers
- LLM-based reranking with Google Gemini Pro
- Balanced recommendations across domains (technical + behavioral)
- RESTful API with FastAPI
- Modern web frontend

## Technology Stack

- **Backend:** FastAPI (Python)
- **LLM:** Google Gemini Pro
- **Embeddings:** Sentence Transformers
- **Frontend:** HTML/CSS/JavaScript
- **Data Processing:** BeautifulSoup4, Pandas
